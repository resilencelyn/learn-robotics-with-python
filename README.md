# 开源机器人学 | Open Robotics

**开源机器人学**是一个帮助机器人学习者入门的开源项目。本项目包含：

1. 高质量的机器人学习教材与文献介绍
2. 有价值的机器人开源项目介绍
3. 帮助学习与理解机器人学 (Robotics) 的python小工具

到目前为止，本项目的内容以作者[@陈金民](https://github.com/chenjm1109)的个人学习与工作经验为主，如有不如人意之处，还请友好指教。
